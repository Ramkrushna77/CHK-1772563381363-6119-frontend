# CHK-1772563381363-6119-frontend

This repository contains the frontend of the MindPulse platform. The frontend provides the user interface where users can interact with the system through chat, voice, and webcam inputs.

The goal of the frontend is to provide a simple and interactive interface for users to communicate with the mental health monitoring system.

Features

User interface for interacting with the MindPulse platform  
Chat interface for communicating with the AI assistant  
Webcam access for facial emotion detection  
Voice input for speech emotion detection  
Display of recommendations and mental health insights  
User authentication using Firebase  

Technology Stack

React.js  
Firebase Authentication  

How it works

The React application allows users to securely sign in using Firebase Authentication.  
After authentication, users can interact with the platform through chat, voice input, and webcam interaction.

User inputs are sent to the backend APIs for processing and analysis.  
The results returned from the backend are then displayed to the user in the form of insights and recommendations.
